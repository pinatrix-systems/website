# 🚀 Pinatrix Systems Website

Official website for **Pinatrix Systems** --- a futuristic technology
ecosystem focused on software, AI, gaming, and digital innovation.

Built with **React + Vite** for fast performance and modern development
experience.

------------------------------------------------------------------------

## ⚙️ Tech Stack

-   React
-   Vite
-   JavaScript (ES6+)
-   CSS

------------------------------------------------------------------------

## 📦 Getting Started

Follow these steps to run the project locally:

### 1. Clone the repository

``` bash
git clone git@github.com:pinatrix-systems/website.git
cd website
```

### 2. Install dependencies

``` bash
npm install
```

### 3. Run the development server

``` bash
npm run dev
```

------------------------------------------------------------------------

## 🌐 Open in Browser

http://localhost:5173
